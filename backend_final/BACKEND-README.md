## Responsibility
This module handles complete backend architecture, including:
- End-to-ENd Integration of Everything
- Authentication including Google OAuth
- Admin Portal and Admin Service
- Frontend Connection and proper secure middleware
- AI servicing and adaption
- Real time data management and auto DB migrations
- Evaluation, Metrics, Cache and Logging
- Security and Privacy First Implementation

## Technologies Used
- FastAPI
- RESTful APIs
- PostgreSQL
- Alembic Migrations
- CORS Middleware 
- External API and Third Party Services
- Python

## How to Run
1. Activate virtual environment
2. Add API keys to `.env`
3. Download requirements.txt
4. run uvicorn app.main:app --reload

## Evaluation
Latency, readability, response quality, and context usage metrics are calculated for performance validation.

## Limitations
- Depends heavily on external LLM APIs
- Requires Internet Connection and stable resources
